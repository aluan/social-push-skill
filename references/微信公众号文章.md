## 快速使用（脚本）

```bash
bash skills/social-push/scripts/weixin-article.sh <文件路径> <标题> [作者] [封面图片路径]
# 示例
bash skills/social-push/scripts/weixin-article.sh /path/to/article.md '我的标题' '张三' /path/to/cover.jpg
```

## 发布微信公众号文章 workflow

1. 打开公众号后台：`agent-browser --headed --profile /tmp/agent-profile open "https://mp.weixin.qq.com/"`
2. 等待加载：`agent-browser wait --load networkidle`
3. 点击"文章"新建：`agent-browser eval "document.querySelector('.new-creation__menu-title').click()"`
4. 等待编辑器加载：`agent-browser wait --load networkidle`
5. 填写标题：`agent-browser fill @e4 "{标题}"`
6. 点击作者框并选择默认作者：`agent-browser click @e5`，然后 `agent-browser find text "{作者名}" click`
7. 点击正文编辑区：`agent-browser click "#ueditor_0"`
8. 输入正文内容：`agent-browser keyboard type "{正文内容}"`（分段用 `agent-browser press "Enter"` 换行）
9. 查看交互：`agent-browser snapshot`
10. 添加封面（hover 触发菜单）：`agent-browser find text "拖拽或选择封面" hover`
11. 点击"从图片库选择"：`agent-browser eval "document.querySelector('#js_cover_null .js_imagedialog').click()"`
12. 上传封面图片：`agent-browser upload "input[type=file][accept*='image/bmp']" "{图片路径}"`
13. 等待上传：`agent-browser wait 2000`
14. 点击"下一步"：`agent-browser click @e26`
15. 点击"确认"完成裁剪：`agent-browser click @e21`
16. 声明原创：`agent-browser find text "未声明" click`，协议已勾选则直接 `agent-browser click @e24`
17. 最后一步根据用户要求选择：
   - 保存草稿：`agent-browser click @e14`（"保存为草稿"按钮）
   - 发布（需用户明确确认）：`agent-browser click @e12`（"发表"按钮）

## 元素参考

| 元素 | 功能 |
|------|------|
| `.new-creation__menu-title` | 新建文章入口 |
| `@e4` | 标题输入框 |
| `@e5` | 作者输入框 |
| `#ueditor_0` | 正文编辑区 |
| `#js_cover_null .js_imagedialog` | 封面"从图片库选择"按钮 |
| `input[type=file][accept*='image/bmp']` | 图片库上传 file input |
| `@e12` / "发表" | 发表按钮 |
| `@e14` / "保存为草稿" | 保存草稿按钮 |

## 注意事项

- 正文编辑器是 UEditor，不是 ProseMirror，需用 `keyboard type` 输入，不能用 `fill`
- 封面上传需先 hover 到封面区域触发菜单，再选择"从图片库选择"
- 图片库弹窗有两个 `input[type=file]`，封面上传用 `accept*='image/bmp'` 的那个
- 原创声明弹窗中协议默认已勾选，直接点确定即可
- 作者框点击后会出现"最近填写"下拉，直接点击选项即可
- 仅保存草稿，由用户手动确认发布
